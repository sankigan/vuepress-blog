> [跳跃训练](https://leetcode.cn/problems/qing-wa-tiao-tai-jie-wen-ti-lcof/description/)

## 题目描述

今天的有氧运动训练内容是在一个长条形的平台上跳跃。

平台有 `num` 个小格子，每次可以选择跳跃 `1` 格或 `2` 格。请返回在训练过程中，学员从起点到最后一个格子有多少种不同的跳跃方案。

由于方案数可能过大，所以请将答案对 `1000000007` 取余，返回最终结果。

**示例**

```
输入：num = 3
输出：3

输入：num = 0
输出：1
```

**提示**

- `0 <= num <= 100`

## 解答

经典斐波那契问题的换皮版。设 `f(i)` 表示跳到第 `i` 格的方案数，到达第 `i` 格的最后一步只可能是：

1. 从第 `i-1` 格跳 `1` 格上来
2. 从第 `i-2` 格跳 `2` 格上来

所以递推式为 `f(i) = f(i-1) + f(i-2)`，初始条件 `f(0) = 1`、`f(1) = 1`。每次累加都对 `1000000007` 取模避免溢出。

### 动态规划

::: details Code
```ts
function trainWays(num: number): number {
  if (num < 2) return 1;

  const MOD = 1000000007;
  const dp = new Array(num + 1).fill(0);
  dp[0] = 1;
  dp[1] = 1;

  for (let i = 2; i <= num; ++i) {
    dp[i] = (dp[i - 1] + dp[i - 2]) % MOD;
  }

  return dp[num];
}
```
:::

### 空间优化

注意到 `dp[i]` 只依赖前两项，可以用两个变量滚动替代整个数组，空间从 O(n) 降到 O(1)。

::: details Code
```ts
function trainWays(num: number): number {
  if (num < 2) return 1;

  const MOD = 1000000007;
  let prev = 1;
  let curr = 1;

  for (let i = 2; i <= num; ++i) {
    const next = (prev + curr) % MOD;
    prev = curr;
    curr = next;
  }

  return curr;
}
```
:::
