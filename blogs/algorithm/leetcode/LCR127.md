## 题目描述

今天的有氧运动训练内容是在一个长条形的平台上跳跃。

平台有 `num` 个小格子，每次可以选择跳跃 `1` 格或 `2` 格。请返回在训练过程中，学员从起点到最后一个格子有多少种不同的跳跃方案。

由于方案数可能过大，所以请将答案对 `1000000007` 取余，返回最终结果。

**示例**

```
输入：num = 3
输出：3

输入：num = 0
输出：1
```

**提示**

- `0 <= num <= 100`
