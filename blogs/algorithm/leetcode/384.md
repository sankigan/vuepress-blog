> [打乱数组](https://leetcode.cn/problems/shuffle-an-array/description/)

## 题目描述

给你一个整数数组 `nums`，设计算法来打乱一个没有重复元素的数组。打乱后，数组的所有排列应该是 **等可能** 的。

实现 `Solution` class:

- `Solution(int[] nums)` 使用整数数组 `nums` 初始化对象
- `int[] reset()` 重设数组到它的初始状态并返回
- `int[] shuffle()` 返回数组随机打乱后的结果

**示例**

```
输入：
["Solution", "shuffle", "reset", "shuffle"]
[[[1, 2, 3]], [], [], []]
输出：
[null, [3, 1, 2], [1, 2, 3], [1, 3, 2]]

解释：
Solution solution = new Solution([1, 2, 3]);
solution.shuffle();    // 打乱数组 [1,2,3] 并返回结果。任何 [1,2,3] 的排列返回的概率应该相同。
                       // 例如，返回 [3, 1, 2]
solution.reset();      // 重设数组到它的初始状态 [1, 2, 3]。返回 [1, 2, 3]
solution.shuffle();    // 随机返回数组 [1, 2, 3] 打乱后的结果。例如，返回 [1, 3, 2]
```

**提示**

- `1 <= nums.length <= 50`
- `-10^6 <= nums[i] <= 10^6`
- `nums` 中的所有元素都是 **唯一的**
- 最多可以调用 `10^4` 次 `reset` 和 `shuffle`

## 解答

### Fisher-Yates 洗牌算法

#### 关键思路

每次 `reset` 都要返回初始状态，因此构造时用 `[...nums]` 备份一份原始数据，调用 `reset` 时从备份中恢复。

`shuffle` 采用经典的 **Fisher-Yates 洗牌算法**：从前往后遍历数组，对每个位置 `i`，从 **未洗牌的剩余区间 `[i, n - 1]`** 中随机选一个下标 `randomIdx`，将其与位置 `i` 的元素交换。

**正确性证明**：每个元素被放到位置 `i` 的概率为 `1/n`（第一步），被放到位置 `i+1` 的概率为 `(1 - 1/n) × 1/(n-1) = 1/n`，以此类推。因此所有 `n!` 种排列出现的概率均等。

**常见误区**：如果对每个位置都在 **整个数组 `[0, n - 1]`** 中随机选下标交换，会得到 `n^n` 种等概率的"操作序列"，但由于 `n^n` 不能被 `n!` 整除，每种排列出现的概率并不均等。

**复杂度**：时间 `O(n)`，空间 `O(n)`（备份数组）。

::: details Code
```ts
class Solution {
  private readonly original: number[];
  private nums: number[];

  constructor(nums: number[]) {
    this.original = [...nums];
    this.nums = [...nums];
  }

  reset(): number[] {
    this.nums = [...this.original];
    return this.nums;
  }

  shuffle(): number[] {
    const n = this.nums.length;
    for (let i = 0; i < n; ++i) {
      // 从剩余未洗牌区间 [i, n - 1] 中随机选一个下标
      const randomIdx = i + Math.floor(Math.random() * (n - i));
      [this.nums[i], this.nums[randomIdx]] = [this.nums[randomIdx], this.nums[i]];
    }
    return this.nums;
  }
}
```
