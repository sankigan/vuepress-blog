> [两两交换链表中的节点](https://leetcode.cn/problems/swap-nodes-in-pairs/description/)

## 题目描述

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在 **不修改节点内部的值** 的情况下完成本题（即，只能进行节点交换）。

**示例**

```
输入：head = [1,2,3,4]
输出：[2,1,4,3]

输入：head = []
输出：[]

输入：head = [1]
输出：[1]
```

**提示**

- 链表中节点的数目在范围 `[0, 100]` 内
- `0 <= Node.val <= 100`

## 解答

### 迭代

#### 关键思路

![](https://s41.ax1x.com/2026/06/25/pmNSeFU.png)

每次交换都需要一个 **前驱指针** 指向待交换的这对节点之前的节点，这样才能在交换后把新的头（原来的第二个节点）接回去。由于头节点本身也要参与交换，引入 `dummy` 哑节点统一处理边界。

对当前这一对 `first → second`，记前驱为 `prev`：

```
prev → first → second → rest
```

交换三步走，**必须按顺序**（否则会丢引用）：

1. `first.next = second.next` —— `first` 指向后续片段 `rest`
2. `second.next = first` —— `second` 指向 `first`，完成本对反转
3. `prev.next = second` —— `prev` 接上交换后的新头

完成后链表变成：

```
prev → second → first → rest
```

下一轮的前驱就是当前的 `first`（它现在排在新对之前），所以 `prev = first` 后继续循环。

**复杂度**：时间 `O(n)`，空间 `O(1)`。

::: details Code
```ts
function swapPairs(head: ListNode | null): ListNode | null {
  const dummy = new ListNode(0, head);
  let prev: ListNode = dummy;

  // 必须保证还有两个节点可交换
  while (prev.next && prev.next.next) {
    const first = prev.next;
    const second = prev.next.next;

    first.next = second.next; // 1. first 指向后续片段
    second.next = first;      // 2. second 指向 first，完成本对反转
    prev.next = second;       // 3. prev 接上新头

    prev = first; // first 此时是下一对的前驱
  }

  return dummy.next;
}
```
:::

### 递归

#### 关键思路

把"两两交换整个链表"分解为：**先交换前两个节点，剩下的子链表递归处理**。

设当前头节点 `head` 与 `head.next` 是要交换的一对：

1. 记 `newHead = head.next`（新的头）
2. 递归处理 `newHead.next` 之后的链表，结果接到 `head.next`
3. `newHead.next = head` 完成本对反转
4. 返回 `newHead` 作为本段链表的新头

**递归出口**：链表为空或只剩一个节点时，无需交换，直接返回。

**复杂度**：时间 `O(n)`，空间 `O(n)`（递归栈深度为 `n / 2`）。

::: details Code
```ts
function swapPairs(head: ListNode | null): ListNode | null {
  // 出口：空链表或单节点无需交换
  if (!head || !head.next) return head;

  const newHead = head.next;
  head.next = swapPairs(newHead.next); // 递归处理后续，挂回 head 之后
  newHead.next = head;                 // 完成本对反转
  return newHead;
}
```
:::
