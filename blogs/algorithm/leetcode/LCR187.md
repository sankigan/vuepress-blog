> [破冰游戏](https://leetcode.cn/problems/yuan-quan-zhong-zui-hou-sheng-xia-de-shu-zi-lcof/description/)

## 题目描述

社团共有 `num` 名成员参与破冰游戏，编号为 `0 ~ num-1`。成员们按照编号顺序围绕圆桌而坐。社长抽取一个数字 `target`，从 0 号成员起开始计数，排在第 `target` 位的成员离开圆桌，且成员离开后从下一个成员开始计数。请返回游戏结束时最后一位成员的编号。

**示例**

```
输入：num = 7, target = 4
输出：1

输入：num = 12, target = 5
输出：0
```

## 解答

#### 关键思路

这是经典的**约瑟夫环**问题。暴力模拟需要每轮线性定位被淘汰者，整体 `O(n²)`；利用数学递推可以压到 `O(n)`。

**核心观察**：每次淘汰一人后，圆圈缩小，剩余成员重新从 `0` 开始编号。如果我们知道 `n - 1` 人游戏的最终胜者在"缩小后圆圈"中的编号，就能推算出他在"原 `n` 人圆圈"中真实的编号。

**推导递推式**：

设 `f(n)` 表示 `n` 人、每次报到第 `target` 的游戏中，最终胜者在原始编号下的位置。

`n` 人报数时，第 `target` 位的人编号为 `(target - 1) % n`（0-Based），淘汰后剩 `n - 1` 人，从下一个人重新开始编号 `0, 1, 2, ...`。

若 `n - 1` 人游戏的胜者在"缩小圆"中编号为 `x`（即 `f(n - 1) = x`），则他在原 `n` 人圆圈中的真实编号为：

```
f(n) = (f(n - 1) + target) % n
```

**边界**：`f(1) = 0`（只剩 1 人时胜者编号固定为 `0`）。

从 `i = 2` 迭代到 `i = n`，每轮用上一结果加 `target` 取模即可，无需额外空间。

#### 复杂度

- 时间：`O(n)`，线性迭代一遍。
- 空间：`O(1)`，只用一个变量滚动更新。

::: details Code
```ts
function iceBreakingGame(num: number, target: number): number {
  let res = 0;
  for (let i = 2; i <= num; ++i) {
    res = (res + target) % i;
  }
  return res;
}
```
:::
