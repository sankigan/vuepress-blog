> [删除排序链表中的重复元素 II](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/description/)

## 题目描述

给定一个已排序的链表的头 `head` ， 删除原始链表中所有重复数字的节点，只留下不同的数字。返回 已排序的链表 。

**示例**

```
输入：head = [1,2,3,3,4,4,5]
输出：[1,2,5]

输入：head = [1,1,1,2,3]
输出：[2,3]
```

**提示**

- 链表中节点数目在范围 `[0, 300]` 内
- `-100 <= Node.val <= 100`
- 题目数据保证链表已经按升序排列

## 解答

### 虚拟头节点 + 遍历

创建一个 `dummy` 节点接在 `head` 前面，用 `prev` 指针追踪"已确认不重复段"的尾节点。每次检查 `prev.next` 是否与后续节点值相同：若相同，内层循环跳过所有相同值节点，再把 `prev.next` 直接指向重复段之后；若不同，`prev` 正常前移。最后返回 `dummy.next`。

::: details Code
```ts
function deleteDuplicates(head: ListNode | null): ListNode | null {
  if (!head) return head;

  const dummy = new ListNode(0, head);
  let prev = dummy;

  while (prev.next) {
    let curr = prev.next;
    if (curr.next && curr.val === curr.next.val) {
      while (curr.next && curr.val === curr.next.val) {
        curr = curr.next;
      }
      prev.next = curr.next;
    } else {
      prev = curr;
    }
  }
  return dummy.next;
}
```
:::
