> [切披萨的方案数](https://leetcode.cn/problems/number-of-ways-of-cutting-a-pizza/description/)

## 题目描述

给你一个 `rows x cols` 大小的矩形披萨和一个整数 `k` ，矩形包含两种字符：`'A'` （表示苹果）和 `'.'` （表示空白格子）。你需要切披萨 `k - 1` 次，得到 `k` 块披萨并送给别人。

切披萨的每一刀，先要选择是按行切还是按列切，然后从前面的若干行或者若干列中选择一个位置切下。在切完最后一块披萨之前，你需要确保每一块披萨都至少包含**一个**苹果。每一刀都将披萨分成两部分：在切割线左边的部分和切割线右边的部分（或者上面的部分和下面的部分），你需要将上面或者左边的部分送给一个人，剩下的留作下一刀继续切。

请你返回切披萨的方案数，由于答案可能是个很大的数字，请你将它对 `10^9 + 7` 取余后返回。

**示例**

```
输入：pizza = ["A..","AAA","..."], k = 3
输出：3
解释：上图展示了三种切披萨的方案。注意每一块披萨都至少包含一个苹果。

输入：pizza = ["A..","AA.","..."], k = 3
输出：1

输入：pizza = ["A..","A..","..."], k = 1
输出：1
```

**提示**

- `1 <= rows, cols <= 50`
- `rows == pizza.length`
- `cols == pizza[i].length`
- `1 <= k <= 10`
- `pizza` 只包含字符 `'A'` 和 `'.'` 。
