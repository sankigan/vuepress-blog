> [重排链表](https://leetcode.cn/problems/reorder-list/description/)

## 题目描述

给定一个单链表 `L` 的头节点 `head` ，单链表 `L` 表示为：

```
L0 → L1 → … → Ln-1 → Ln
```

请将其重新排列后变为：

```
L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → …
```

不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

**示例**

```
输入：head = [1,2,3,4]
输出：[1,4,2,3]

输入：head = [1,2,3,4,5]
输出：[1,5,2,4,3]
```

**提示**

- 链表的长度范围为 `[1, 5 * 10^4]`
- `1 <= node.val <= 1000`

## 解答

观察目标排列 `L0 → Ln → L1 → Ln-1 → …`，等价于把链表从中点切成两段，将后半段反转，再与前半段交错合并。整体三步：

1. **找中点**：快慢指针，`fast` 每次走两步、`slow` 每次走一步，循环终止时 `slow` 落在前半段的末尾（偶数长度时落在前半段最后一个节点）
2. **反转后半段**：从 `slow.next` 起做标准的链表反转，同时将 `slow.next` 置空切断前后半连接
3. **交错合并**：前半段 `p1` 与反转后的后半段 `p2` 轮流穿插，每次把 `p2` 接到 `p1` 后面，再让 `p1`、`p2` 各前进一步

举例 `[1,2,3,4,5]`：

- 中点切分：`[1,2,3]` 和 `[4,5]`
- 后半反转：`[5,4]`
- 交错合并：`1 → 5 → 2 → 4 → 3`

::: details Code
```ts
function reverseList(head: ListNode | null): ListNode | null {
  let prev: ListNode | null = null;
  let curr = head;
  while (curr) {
    const next = curr.next;
    curr.next = prev;
    prev = curr;
    curr = next;
  }
  return prev;
}

function merge(p1: ListNode | null, p2: ListNode | null): void {
  while (p1 && p2) {
    const n1 = p1.next;
    const n2 = p2.next;
    p1.next = p2;
    p2.next = n1;
    p1 = n1;
    p2 = n2;
  }
}

function reorderList(head: ListNode | null): void {
  if (!head || !head.next) return;

  // 1. 快慢指针找中点
  let slow = head;
  let fast = head;
  while (fast.next && fast.next.next) {
    slow = slow.next!;
    fast = fast.next.next;
  }

  // 2. 切分并反转后半段
  const second = reverseList(slow.next);
  slow.next = null;

  // 3. 交错合并
  merge(head, second);
}
```
