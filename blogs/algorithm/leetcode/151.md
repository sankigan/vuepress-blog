> [反转字符串中的单词](https://leetcode.cn/problems/reverse-words-in-a-string/description/)

## 题目描述

给你一个字符串 `s` ，请你反转字符串中**单词**的顺序。

**单词**是由非空格字符组成的字符串。`s` 中使用至少一个空格将字符串中的**单词**分隔开。

返回**单词**顺序颠倒且**单词**之间用单个空格连接的结果字符串。

**注意**：输入字符串 `s` 中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

**示例**

```
输入：s = "the sky is blue"
输出："blue is sky the"

输入：s = "  hello world  "
输出："world hello"
解释：反转后的字符串中不能存在前导空格和尾随空格。

输入：s = "a good   example"
输出："example good a"
解释：如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。
```

**提示**

- `1 <= s.length <= 10^4`
- `s` 包含英文大小写字母、数字和空格 `' '`
- `s` 中**至少存在一个**单词

**进阶**：如果字符串在你使用的编程语言中是一种可变数据类型，请你使用 `O(1)` 额外空间复杂度的**原地**解法。

## 解答

### 从右向左扫描 + 单词切片

#### 关键思路

题目本质是两件事：**反转单词顺序** + **规范化空格**（去掉首尾空格、单词间多空格折叠成一个）。

与其先 `trim` + `split(/\s+/)` + `reverse` + `join` 分多步处理空格，更直观的做法是 **从右向左扫描原串**，遇到单词整段取出按出现顺序压入结果数组，空格直接跳过——天然完成了「反转顺序」和「忽略多余空格」两件事：

1. 用一个游标 `scan` 从字符串末尾向前走；
2. 遇到非空格字符，启动内层循环把这段连续非空格字符**拼成一个单词**（注意是从后往前拼，所以新字符要拼在已得片段的**前面**，保证单词内部字符顺序不变）；
3. 整段读完后 push 进 `res`，此时 `res` 中单词的出现顺序正好就是原串从右往左的顺序，即反转结果；
4. 遇到空格直接 `scan--` 跳过，无论中间有多少连续空格，都不会进入收集分支，自然完成空格折叠；
5. 最终 `res.join(' ')` 用单空格连接即可。

#### 复杂度

- 时间：`O(n)`，每个字符被外层或内层游标各访问一次。
- 空间：`O(n)`，用于存放结果数组与拼接的单词片段；若要做到 `O(1)` 额外空间（进阶要求），需先整体反转字符再逐单词反转，且字符串需可变。

::: details Code
```ts
function reverseWords(s: string): string {
  const res: string[] = [];
  let scan = s.length - 1;

  while (scan >= 0) {
    // 跳过空格段，定位到下一个单词的末尾
    if (/\s/.test(s[scan])) {
      scan--;
      continue;
    }

    // 收集一个完整单词：从右往左读，所以新字符拼在已得片段的前面
    let seg = '';
    while (scan >= 0 && !/\s/.test(s[scan])) {
      seg = s[scan--] + seg;
    }
    res.push(seg);
  }

  return res.join(' ');
}
```
