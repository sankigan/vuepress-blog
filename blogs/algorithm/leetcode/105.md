> [从前序与中序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/)

## 题目描述

给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的 **先序遍历**，`inorder` 是同一棵树的 **中序遍历**，请构造二叉树并返回其根节点。

**示例**

```
输入：preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出：[3,9,20,null,null,15,7]

输入：preorder = [-1], inorder = [-1]
输出：[-1]
```

**提示**

- `1 <= preorder.length <= 3000`
- `inorder.length == preorder.length`
- `-3000 <= preorder[i], inorder[i] <= 3000`
- `preorder` 和 `inorder` 均 **无重复** 元素
- `inorder` 均出现在 `preorder`
- `preorder` **保证** 为二叉树的前序遍历序列
- `inorder` **保证** 为二叉树的中序遍历序列

## 解答

### 关键思路

利用前序遍历和中序遍历的特性递归构造二叉树：

- **前序遍历**：`[根 | 左子树 | 右子树]`，第一个元素一定是当前子树的根
- **中序遍历**：`[左子树 | 根 | 右子树]`，根的位置将数组划分为左右子树

每次从 `preorder` 取出根节点，在 `inorder` 中定位根的位置，左侧即为左子树的中序序列、右侧为右子树的中序序列，再据此划分 `preorder` 的左右部分，递归构造即可。

### 朴素递归

每次递归通过 `findIndex` 在 `inorder` 中线性查找根的位置，并用 `slice` 切分新的子数组。逻辑直观但效率较低。

**复杂度**：时间 `O(n²)`（每层 `findIndex` + `slice`），空间 `O(n²)`（每次都创建新数组）。

::: details Code
```ts
function buildTree(preorder: number[], inorder: number[]): TreeNode | null {
  if (!preorder.length || !inorder.length) return null;

  const rootVal = preorder[0];
  const rootIdx = inorder.findIndex(d => d === rootVal);
  const leftInOrder = inorder.slice(0, rootIdx);
  const rightInOrder = inorder.slice(rootIdx + 1);

  const root = new TreeNode(rootVal);
  root.left = buildTree(preorder.slice(1, 1 + leftInOrder.length), leftInOrder);
  root.right = buildTree(preorder.slice(1 + leftInOrder.length), rightInOrder);

  return root;
};
```
:::

### 哈希表优化

针对朴素递归的两个瓶颈分别优化：

1. **用哈希表预存 `inorder` 中每个值的索引**，将根定位从 `O(n)` 降为 `O(1)`
2. **不再切分数组**，改用 `[start, end]` 区间表示当前子树在 `inorder` 中的范围
3. **用全局指针 `preIdx`** 顺序读取前序序列，每次递归取一个根节点后自增

由于前序遍历总是 "根 → 左子树 → 右子树" 的顺序，递归时先构造左子树会自然消费完左子树对应的所有前序节点，`preIdx` 接下来指向的就是右子树的根，所以无需显式切分 `preorder`。

**复杂度**：时间 `O(n)`，空间 `O(n)`（哈希表 + 递归栈）。

::: details Code
```ts
function buildTree(preorder: number[], inorder: number[]): TreeNode | null {
  const map = new Map<number, number>();
  inorder.forEach((d, i) => map.set(d, i));

  let preIdx = 0;

  const build = (start: number, end: number): TreeNode | null => {
    if (start > end) return null;

    const rootVal = preorder[preIdx++];
    const rootIdx = map.get(rootVal) as number;

    const root = new TreeNode(rootVal);
    root.left = build(start, rootIdx - 1);
    root.right = build(rootIdx + 1, end);

    return root;
  };

  return build(0, inorder.length - 1);
};
```
