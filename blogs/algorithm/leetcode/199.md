> [二叉树的右视图](https://leetcode.cn/problems/binary-tree-right-side-view/description/)

## 题目描述

给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

**示例**

```
输入：[1,2,3,null,5,null,4]
输出：[1,3,4]

输入：[1,2,3,4,null,null,null,5]
输出：[1,3,4,5]

输入：[1,null,3]
输出：[1,3]

输入：[]
输出：[]
```

## 解答

### BFS 层序遍历

::: details Code
```ts
function rightSideView(root: TreeNode | null): number[] {
  const queue = [root];
  const res: number[] = [];
  if (!root) return res;

  while (queue.length) {
    const curLvlNode = [];
    let curLvlSize = queue.length;

    while (curLvlSize--) {
      const node = queue.shift();
      curLvlNode.push(node.val);
      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }

    res.push(curLvlNode[curLvlNode.length - 1]);
  }

  return res;
};
```
