> [最小路径和](https://leetcode.cn/problems/minimum-path-sum/description/)

## 题目描述

给定一个包含非负整数的 `m x n` 网格 `grid`，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

**说明**：每次只能向下或者向右移动一步。

**示例**

```
输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：因为路径 1→3→1→1→1 的总和最小。

输入：grid = [[1,2,3],[4,5,6]]
输出：12
```

**提示**

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 200`
- `0 <= grid[i][j] <= 200`

## 解答

### 动态规划

#### 思路

1. 定义状态
   - `dp[i][j]` 表示从左上角 `(0, 0)` 走到位置 `(i, j)` 的最小路径和
2. 状态转移
   - 由于每一步只能向下或向右，所以 `(i, j)` 只能从 **上方 `(i - 1, j)`** 或 **左方 `(i, j - 1)`** 走过来
   - 取这两条路径的较小值，再加上当前格子的代价：
     - `dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]`
3. 初始化
   - `dp[0][0] = grid[0][0]`
   - 第一行只能从左边走过来：`dp[0][j] = dp[0][j - 1] + grid[0][j]`
   - 第一列只能从上边走过来：`dp[i][0] = dp[i - 1][0] + grid[i][0]`
4. 计算结果
   - 答案就是 `dp[m - 1][n - 1]`

**复杂度**：时间 `O(m × n)`，空间 `O(m × n)`。

::: details Code
```ts
function minPathSum(grid: number[][]): number {
  const m = grid.length;
  const n = grid[0].length;
  const dp: number[][] = Array.from({ length: m }, () => new Array(n).fill(0));

  dp[0][0] = grid[0][0];
  // 第一列：只能从上方走来
  for (let i = 1; i < m; ++i) dp[i][0] = dp[i - 1][0] + grid[i][0];
  // 第一行：只能从左方走来
  for (let j = 1; j < n; ++j) dp[0][j] = dp[0][j - 1] + grid[0][j];

  for (let i = 1; i < m; ++i) {
    for (let j = 1; j < n; ++j) {
      dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
    }
  }

  return dp[m - 1][n - 1];
}
```
:::

#### 降维优化

观察转移方程 `dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]`，`dp[i][j]` 只依赖：

- **正上方** `dp[i - 1][j]`（上一行同列）
- **正左方** `dp[i][j - 1]`（同一行前一列）

不依赖左上角，所以可以用一维数组 `dp[j]` 压缩：

- 处理第 `i` 行前，`dp[j]` 保存的是 **上一行** 的 `dp[i - 1][j]`
- 计算 `dp[j] = min(dp[j], dp[j - 1]) + grid[i][j]`：
  - 等号右边的 `dp[j]` 还没被覆盖，正是上一行的值（上方）
  - 等号右边的 `dp[j - 1]` 已在本轮被更新为当前行的值（左方）
- 所以 `j` 必须 **从小到大** 遍历，这点和 [718 题](./718) 依赖左上角的场景刚好相反

每行第一列要单独累加：`dp[0] += grid[i][0]`（只能从上方过来）。

**复杂度**：时间 `O(m × n)`，空间 `O(n)`。

::: details Code
```ts
function minPathSum(grid: number[][]): number {
  const m = grid.length;
  const n = grid[0].length;
  const dp: number[] = new Array(n).fill(0);

  // 初始化第一行
  dp[0] = grid[0][0];
  for (let j = 1; j < n; ++j) dp[j] = dp[j - 1] + grid[0][j];

  for (let i = 1; i < m; ++i) {
    dp[0] += grid[i][0]; // 第一列：只能从上方过来
    for (let j = 1; j < n; ++j) {
      // dp[j]：上一行同列（上方）；dp[j - 1]：当前行前一列（左方）
      dp[j] = Math.min(dp[j], dp[j - 1]) + grid[i][j];
    }
  }

  return dp[n - 1];
}
```
