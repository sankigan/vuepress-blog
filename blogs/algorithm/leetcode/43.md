> [字符串相乘](https://leetcode.cn/problems/multiply-strings/description/)

## 题目描述

给定两个以字符串形式表示的非负整数 `num1` 和 `num2`，返回 `num1` 和 `num2` 的乘积，它们的乘积也表示为字符串形式。

**注意**：不能使用任何内置的 BigInteger 库，或直接将输入转换为整数。

**示例**

```
输入: num1 = "2", num2 = "3"
输出: "6"

输入: num1 = "123", num2 = "456"
输出: "56088"
```

**提示**

- `1 <= num1.length, num2.length <= 200`
- `num1` 和 `num2` 只能由数字组成。
- `num1` 和 `num2` 都不包含任何前导零，除了数字 0 本身。

**进阶**：你可以不使用内置 BigInteger 库或直接将输入转换为整数来解决这个问题吗？

## 解答

### 竖式乘法模拟

#### 关键思路

模拟手算竖式乘法的过程。两个数长度分别为 `m` 和 `n`，乘积最多为 `m + n` 位，因此创建一个长度为 `m + n` 的数组存放每一位的结果。

核心规律：`num1[i] * num2[j]` 的结果会贡献到结果数组的第 `i + j` 位（高位进位）和第 `i + j + 1` 位（当前位），从低位向高位逐步累加并处理进位，最后去除前导零即可。

:::: details Code
```ts
function multiply(num1: string, num2: string): string {
  const m = num1.length;
  const n = num2.length;
  const ans = new Array(m + n).fill(0);

  for (let i = m - 1; i >= 0; --i) {
    for (let j = n - 1; j >= 0; --j) {
      const mul = +num1[i] * +num2[j] + ans[i + j + 1];
      ans[i + j + 1] = mul % 10;
      ans[i + j] += Math.floor(mul / 10);
    }
  }

  let res = ans.join('');
  while (res.startsWith('0')) res = res.slice(1);
  return res || '0';
};
```
