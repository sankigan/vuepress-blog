> [删除链表的倒数第 N 个结点](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/description/)

## 题目描述

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**示例**

```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

输入：head = [1], n = 1
输出：[]

输入：head = [1,2], n = 1
输出：[1]
```

**提示**

- 链表中结点的数目为 `sz`
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

**进阶**：你能尝试使用一趟扫描实现吗？

## 解答

### 关键思路

要删除倒数第 `n` 个节点，需要定位到 **倒数第 `n + 1` 个节点**（即待删除节点的前驱），再修改其 `next` 指针跳过目标节点即可。

使用 **快慢双指针** 一趟扫描完成：

1. 引入虚拟头节点 `dummy`，统一处理"删除头节点"的边界情况
2. `fast` 先从 `dummy` 出发走 `n + 1` 步，使 `fast` 与 `slow` 之间相隔 `n + 1` 个节点
3. `fast` 和 `slow` 同时前进，当 `fast` 走到 `null` 时，`slow` 恰好停在待删除节点的前驱
4. 执行 `slow.next = slow.next.next` 完成删除

**复杂度**：时间 `O(sz)`，空间 `O(1)`。

::: details Code
```ts
function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
  const dummy = new ListNode(0, head);
  let fast: ListNode | null = dummy;
  let slow: ListNode = dummy;

  // fast 先走 n + 1 步，使 fast 与 slow 之间相隔 n + 1 个节点
  for (let i = 0; i <= n; ++i) {
    fast = fast!.next;
  }

  // 同步前进，fast 到达末尾时 slow 指向待删除节点的前驱
  while (fast) {
    slow = slow.next!;
    fast = fast.next;
  }

  slow.next = slow.next!.next;

  return dummy.next;
};
```
